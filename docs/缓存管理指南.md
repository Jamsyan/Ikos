# 缓存管理指南

本文档介绍 Ikos 的模型缓存管理机制。

## 核心功能

Ikos 缓存管理器提供以下功能：

1. **统一的缓存目录结构**
2. **自动清理无用文件**（README/LICENSE 等）
3. **版本管理**
4. **完整性验证**
5. **空间统计**
6. **清理工具**

## 缓存目录结构

```
~/.cache/ikos/
├── models/
│   └── modelscope/          # 魔塔社区模型
│       ├── damo/
│       │   └── nlp_csanmt_translationzh2en/
│       │       └── master/
│       │           ├── config.json
│       │           ├── model.pt
│       │           ├── tokenizer.json
│       │           └── metadata.json  # 元数据
│       └── qwen/
│           └── qwen-7b/
│               └── master/
├── temp/                    # 临时目录
└── logs/                    # 日志目录
```

### 目录说明

| 目录 | 说明 |
|------|------|
| `models/modelscope/` | 魔塔社区模型 |
| `models/huggingface/` | Hugging Face 模型（保留接口） |
| `temp/` | 临时下载文件 |
| `logs/` | 下载日志 |

## 自动清理机制

下载模型时会自动清理以下文件：

### 清理的文件类型

- `README.md`, `README_zh.md` - 说明文档
- `LICENSE`, `LICENSE.txt` - 许可证
- `NOTICE` - 声明文件
- `.gitignore` - Git 配置
- 所有 `*.md` 文件

### 保留的核心文件

- `*.json` - 配置文件（config.json 等）
- `*.bin`, `*.pt`, `*.pth` - PyTorch 模型
- `*.safetensors` - SafeTensors 格式
- `*.onnx` - ONNX 格式
- `*.txt`, `*.vocab` - 词汇表
- `*.model` - SentencePiece 模型


输出示例：

#### 3. 清理指定模型

```bash
ikos-cache clean --model damo/nlp_csanmt_translationzh2en
```

#### 4. 清理旧模型


```bash

```


## Python API 使用

### 获取缓存管理器

```python
from ikos.utils import get_cache_manager

# 获取全局实例
manager = get_cache_manager()

# 或指定缓存目录
manager = get_cache_manager("/path/to/cache")
```

### 查看统计信息

```python
stats = manager.get_cache_stats()

print(f"总模型数：{stats['total_models']}")
print(f"总大小：{manager._format_size(stats['total_size'])}")
```

### 清理缓存

```python
# 清理指定模型
manager.clear_cache(model_id="damo/nlp_csanmt_translationzh2en")

# 清理 30 天前的模型
manager.clear_cache(older_than_days=30)

# 清理所有
manager.clear_cache()
```

### 验证完整性

```python
model_path = manager.get_model_path("damo/nlp_csanmt_translationzh2en")
integrity = manager.verify_integrity(model_path)

if integrity["valid"]:
    print("✅ 完整性验证通过")
else:
    print("❌ 完整性验证失败")
    for missing in integrity["missing_files"]:
        print(f"缺失：{missing}")
```

## 最佳实践

### 1. 定期清理

```bash
# 每周清理一次 30 天前的模型
0 0 * * 0 ikos-cache clean --days 30
```

### 2. 下载前检查缓存

```python
from ikos.utils import get_cache_manager

manager = get_cache_manager()
model_path = manager.get_model_path("damo/nlp_csanmt")

if model_path.exists():
    print("模型已缓存，无需重新下载")
else:
    print("需要下载模型")
```

### 3. 验证重要模型

对于关键任务，下载后验证完整性：

```python
integrity = manager.verify_integrity(model_path)
if not integrity["valid"]:
    print("模型不完整，重新下载")
    # 重新下载逻辑
```

## 常见问题

### Q1: 缓存目录在哪？

默认：`~/.cache/ikos/`

可以通过 `--cache-dir` 参数指定：
```bash
ikos-cache stats --cache-dir /path/to/custom/cache
```

### Q2: 如何彻底清理缓存？

```bash
# 方法 1：命令行
ikos-cache clean --all

# 方法 2：手动删除
rm -rf ~/.cache/ikos/
```

### Q3: 缓存会占用多少空间？

取决于下载的模型数量，通常：
- 小模型（<1GB）：5-10 个
- 大模型（7B+）：2-3 个

建议定期清理不用的模型。

### Q4: 可以移动缓存目录吗？

可以，但需要：
1. 复制整个 `~/.cache/ikos/` 目录
2. 使用 `--cache-dir` 指定新位置

### Q5: 为什么清理了 README 等文件？

这些文件对模型运行无用，只会占用空间。如需查看说明，可以访问魔塔社区网站。
