## 技术亮点

### 1. 硬件自适应

```python
# 自动检测并分级
hardware = detect_hardware()
# 输出：
# GPU: NVIDIA RTX 3060 (12.0GB)
# 硬件分级：基础场景
# 推荐引擎：混合模式
```

### 2. 智能量化推荐

```python
# 根据硬件和模型自动推荐
config = auto_recommend_quantization("qwen3.5:7b", hardware)
# 8GB 显存 -> INT8
# 4GB 显存 -> NF4
# 16GB 显存 -> FP16
```

### 3. 显存精细化管理

```python
# VRAM 池化 + 优先级调度
vram = VRAMManager.from_hardware(hardware)
vram.allocate(4.0, "qwen-7b", Priority.P0)  # P0 可抢占
```

### 4. CPU-GPU 混合推理

```python
# 自动启用，突破显存限制
# 4GB 显存可运行 7B 模型（NF4 量化）
model_loader.load_model("Qwen/Qwen2.5-7B-Instruct")
```

### 5. 引擎热切换

```python
# 一键切换不同引擎
switcher = EngineSwitcher()
engine = switcher.switch_to(EngineType.NATIVE)
```

## 使用示例

### 基础使用

```python
from ikos.core import create_native_engine

engine = create_native_engine(quantization="auto")

response = engine.call(
    prompt="请介绍一下傅里叶变换",
    model="Qwen/Qwen2.5-7B-Instruct",
    max_tokens=1024
)

print(response.content)
```

### 多模型投票

```python
result = engine.vote(
    prompt="解释量子力学",
    models=[
        "Qwen/Qwen2.5-7B-Instruct",
        "Qwen/Qwen2.5-3B-Instruct",
        "deepseek-ai/DeepSeek-R1-Distill-Qwen-7B"
    ]
)

print(f"获胜：{result.winner_model}")
```
